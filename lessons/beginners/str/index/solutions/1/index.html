Řetězce – Řešení [1]

print('Čokoláda'[-1])  # → a
print('Čokoláda'[-2])  # → d
print('Čokoláda'[-3])  # → á
print('Čokoláda'[-4])  # → l

Záporná čísla vybírají písmenka od konce.
